# MCA MAH CAT Mathematics Prep

A comprehensive, interactive learning platform designed to help students master the Mathematics and Statistics syllabus for the MCA MAH CAT entrance exam. This application combines detailed theoretical content with immersive, real-time visualizers to provide a deep understanding of complex concepts.

## Features

- **Comprehensive Syllabus Coverage**: Detailed modules for Algebra, Coordinate Geometry, Differential Equations, Trigonometry, Probability & Statistics, Sets, Arithmetic, and Mensuration.
- **Interactive Visualizers**:
  - **Quadratic Solver**: Real-time graphing and root calculation.
  - **Coordinate Geometry**: Interactive distance, midpoint, and slope tools.
  - **Trigonometry**: Dynamic Unit Circle visualizer.
  - **Probability**: Interactive Normal Distribution curve.
  - **Sets**: Real-time Venn Diagram for set operations.
  - **Mensuration**: 3D shape calculators for Cube, Cylinder, and Sphere.
- **Professional Math Rendering**: High-quality mathematical typography using **KaTeX**.
- **Immersive UI**: "Masterclass" aesthetic with smooth animations (Framer Motion) and a clean, modern design (Tailwind CSS).
- **Responsive Design**: Fully optimized for all device sizes.

## Tech Stack

- **Frontend Framework**: React
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Math Rendering**: KaTeX

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd mca-maths
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the Application

To start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173`.

### Building for Production

To create a production build:

```bash
npm run build
```

The built files will be in the `dist` directory.
